# Voice to Text AI Audio Annotation System

A Flask-based project that converts audio into text using Whisper, generates summaries, and creates a knowledge graph from the transcript.

## Features
- Upload audio files
- Convert speech to text using Whisper
- Generate transcript summary
- Create a simple knowledge graph
- Flask web interface

## Tech Stack
- Python
- Flask
- OpenAI Whisper
- Transformers
- NetworkX
- HTML/CSS
